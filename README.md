# The Little Empress — Grimoire of the Digital Worlds

A ritualized, parent-friendly guide to setting up a **Wii** and **iPad** for young children, blending **Kongo cosmology** with practical screen-time controls and **multilingual co-play**.

---

## 🎮 Quick Start

### Visit the Live Grimoire
**[https://NounStream.github.io/little-empress-grimoire/](https://NounStream.github.io/little-empress-grimoire/)**

### On an iPad (iOS 9.3.5+)
1. Open **Safari** and visit the link above.
2. Once the page loads, tap **Share** → **Add to Home Screen** → tap **Add**.
3. A new icon appears; tap it to open the grimoire full-screen.
4. Progress is saved locally on your device.

### On Mac or Computer
1. Visit the link above in any browser.
2. Click "Mark Quest Complete" as you finish each quest.
3. Progress updates and persists across page reloads.

## 🛠 How to Customize

The grimoire is a single **HTML file** with embedded CSS and JavaScript. You can:

1. **Edit text directly** in `index.html` using any text editor.
2. **Change colors** by modifying the `<style>` section.
3. **Swap languages** using the 4-slot language template.
4. **Add new quests** by copying quest card structures.
5. **No build process required**—just edit and save.

### All progress is stored locally in the browser (no server needed).

## 🎯 For Parents & Guardians

### Before You Start
1. Read through the entire grimoire once (takes ~15 minutes).
2. Gather your devices: Wii, iPad, and any game discs.
3. Set aside **1–2 hours** to work through the setup quests.

### During Each Quest
1. Follow the numbered steps carefully.
2. Use the glossary explanations for Kongo terms.
3. Check the "Real-World Echo" for co-play ideas.
4. Mark the quest complete when done.

### Safety Reminders
- **ROM & Emulation**: Only dump games you own physically; do not download pirated ROMs.  
  Laws vary by country—this is general orientation, not legal advice.
- **Parental Controls**: Always set a passcode different from the device PIN.
- **Screen Time**: Pediatric guidance suggests 30–60 minutes per day for young children.  
  No screens at meals or before sleep.
