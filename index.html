⚠ Setup Instructions

The full interactive grimoire file needs to be uploaded to this repository. To get started:

  1. Download your little-empress-grimoire-v3-progress.html file
  2. Visit the GitHub repository
  3. Upload the HTML file and rename it to index.html
  4. In the repository Settings → Pages, enable GitHub Pages
  5. Access the grimoire at: https://NounStream.github.io/little-empress-grimoire/
